One Eye is Enough

Electronic Engineer / Software Engineer, Hardware and Software, Python, Arduino · All design, software and hardware implementation
Collaborating with Richard Hards on a number of installations at RCA, Barbican, Beaconsfields and others.
A sound/sculpture installation that from an electronics standpoint consists of around 80 motors controlled from a standalone Python app that launches all sorts of patterns. Each motor hits against part of the environment, be it a window or metal beam, to create a huge audio scape.
More can be seen at richardhards.com

LED Light Wand

80 RGB LED strip that displays bitmaps sent via WiFi
Electronic Engineer / Software Engineer, Arduino, Java · A 1.5 meter tube that stores bitmaps sent via WiFi.
A display on the Wand lets you select an image and then trigger.

The user then holds the Wand vertically and slowly drags the Wand sideways with the LEDs pointing towards a camera taking a long exposure shot for miraculous results.

Commissioned by a photography events company in LA.

Full design and construction of Hardware and Standalone app for Windows/Mac.
